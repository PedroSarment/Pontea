Introduction

Base URL: http://localhost:8000

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Activities

This function allows you to filter activities based on various criteria, including text in the title or description, age group, area, level, created by user, and the presence of multimedia resources or visual instructions.

Retrieve all activities with related information.

Example request:
curl --request GET \
    --get "http://localhost:8000/api/activity" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --header "Authorization: Bearer $2y$10$mhuGD2BQ6WZYTZcpPxwTHOIj/aQlmgG9ahXn66BZQ.GBmbGB7gggi"
const url = new URL(
    "http://localhost:8000/api/activity"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
    "Authorization": "Bearer $2y$10$mhuGD2BQ6WZYTZcpPxwTHOIj/aQlmgG9ahXn66BZQ.GBmbGB7gggi",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

Filter activities based on request parameters.

Example request:
curl --request GET \
    --get "http://localhost:8000/api/activity/filter?text=cumque&age_group_id=6&area_id=18&level_id=1&created_by=2&has_multimedia_resources=&has_visual_instructions=" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --header "Authorization: Bearer $2y$10$mhuGD2BQ6WZYTZcpPxwTHOIj/aQlmgG9ahXn66BZQ.GBmbGB7gggi"
const url = new URL(
    "http://localhost:8000/api/activity/filter"
);

const params = {
    "text": "cumque",
    "age_group_id": "6",
    "area_id": "18",
    "level_id": "1",
    "created_by": "2",
    "has_multimedia_resources": "0",
    "has_visual_instructions": "0",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
    "Authorization": "Bearer $2y$10$mhuGD2BQ6WZYTZcpPxwTHOIj/aQlmgG9ahXn66BZQ.GBmbGB7gggi",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

Request      

GET api/activity/filter

Headers

Content-Type      

Example: application/json

Accept      

Example: application/json

Authorization      

Example: Bearer $2y$10$mhuGD2BQ6WZYTZcpPxwTHOIj/aQlmgG9ahXn66BZQ.GBmbGB7gggi

Query Parameters

text   string  optional  

The text to search in the title or description of the activity (optional). Example: cumque

age_group_id   integer  optional  

The ID of the age group to filter activities by a specific age group (optional). Example: 6

area_id   integer  optional  

The ID of the area to filter activities by a specific area (optional). Example: 18

level_id   integer  optional  

The ID of the level to filter activities by a specific level (optional). Example: 1

created_by   integer  optional  

The ID of the user who created the activity to filter activities created by a specific user (optional). Example: 2

has_multimedia_resources   boolean  optional  

Defines whether the activity should have multimedia resources (true) or not (false) (optional). Example: false

has_visual_instructions   boolean  optional  

Defines whether the activity should have visual instructions (true) or not (false) (optional). Example: false
